Proyek statistika kami menggunakan model Ridge Regression untuk memprediksi suhu berdasarkan dataset BMKG tahun 2017 hingga 2023 dari Kaggle untuk wilayah Kota Semarang, serta API BMKG untuk mendapatkan data-data yang berkaitan dengan cuaca pada hari ini untuk memprediksi satu minggu ke depan.

  • Variabel dependen

    • Output dari model machine learning kami terdiri dari hari dan tanggal, suhu minimal, suhu maksimal, suhu rata-rata, dan kelembapan

  • Variabel independen

    • API BMKG sebagai variabel independen untuk memengaruhi model machine learning kami.
